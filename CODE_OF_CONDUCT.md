# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Provide helpful feedback.
- Focus on technical issues, not people.

## Unacceptable Behavior

- Harassment, discrimination, or hate speech.
- Personal attacks or insults.
- Trolling, intimidation, or disruptive behavior.
- Publishing private information without consent.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and PRs that violate this Code of Conduct.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, and discussions related to the repository.

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers through repository communication channels.
