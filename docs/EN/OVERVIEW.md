# Overview

Waveon is a Windows desktop music application focused on a **local-first** workflow.

The app helps you:

- build a personal music library on disk
- import tracks from SoundCloud URLs
- organize tracks into playlists
- play music with queue controls and timeline seek

Waveon stack:

- Electron (desktop shell)
- React + TypeScript (UI)
- Zustand (client state)
- SQLite via better-sqlite3 (local data)
- yt-dlp + ffmpeg (download/transcode pipeline)
