# Features

## Library

- Persistent local library backed by SQLite
- Track metadata storage: title, artist, duration, cover, source URL, file path
- Track deletion with playlist relation cleanup

## Playlist Management

- Create playlists with optional custom cover
- Rename and delete playlists
- Add/remove tracks in playlists
- Preserve playlist ordering

## Import & Download

- Import playlist tracks from SoundCloud playlist URLs
- Download single tracks from supported URLs
- Automatic metadata extraction before download
- MP3 conversion via ffmpeg

## Playback UX

- Bottom player with play/pause, previous/next
- Timeline seek with drag interaction
- Volume control and default volume state
- Queue-aware playback from library or selected playlist

## Interface

- Page-based navigation (Home, Download, Library, Playlists, etc.)
- Animated UI with optional animation toggle
- Compact mode and confirmation preferences in settings
