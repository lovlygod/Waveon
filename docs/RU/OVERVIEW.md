# Обзор

Waveon — Windows desktop-приложение для создания и прослушивания локальной музыкальной библиотеки.

Основные направления:

- управление локальными треками
- работа с плейлистами
- импорт треков по ссылкам
- современный интерфейс и удобное воспроизведение

Технологический стек:

- Electron
- React + TypeScript
- Zustand
- SQLite (better-sqlite3)
- yt-dlp + ffmpeg
