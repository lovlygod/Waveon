# Установка

## Требования

- Windows 10/11
- Node.js 18+
- npm
- установленный ffmpeg

## Шаги

1. Клонируйте репозиторий.
2. Установите зависимости:

```bash
npm install
```

3. Убедитесь, что ffmpeg доступен в PATH.

4. (Опционально) проверьте типы:

```bash
npm run typecheck
```

## Примечания

- Локальные данные приложения сохраняются в:
  - `data/music`
  - `data/covers`
  - `database/`
- При первом запуске недостающие папки создаются автоматически.
