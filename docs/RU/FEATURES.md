# Возможности

## Библиотека

- Локальная музыкальная библиотека на SQLite
- Хранение метаданных трека: название, артист, длительность, обложка, источник, путь к файлу
- Удаление треков с очисткой связей в плейлистах

## Плейлисты

- Создание плейлистов (с опциональной обложкой)
- Переименование и удаление плейлистов
- Добавление/удаление треков
- Сохранение порядка треков

## Импорт и загрузка

- Импорт треков из плейлистов SoundCloud
- Загрузка отдельных треков по URL
- Извлечение метаданных перед скачиванием
- Конвертация в MP3 через ffmpeg

## Воспроизведение и UI

- Нижний плеер: play/pause, prev/next, таймлайн, громкость
- Работа с очередью из библиотеки и плейлистов
- Современный интерфейс с анимациями
- Настройки отображения и поведения
