# Решение проблем

## ffmpeg не найден

- Убедитесь, что ffmpeg установлен.
- Добавьте ffmpeg в PATH или задайте путь к ffmpeg в окружении.

## Ошибка импорта

- Проверьте корректность ссылки SoundCloud.
- Используйте публичный URL плейлиста.

## Не найден итоговый mp3-файл

- Проверьте наличие `yt-dlp`.
- Убедитесь, что ffmpeg доступен в том же окружении, где запущен Electron.

## Ошибки сборки

- Выполните `npm run typecheck`.
- При необходимости удалите `node_modules` и установите зависимости заново.

## Ссылки открываются внутри приложения

- Проверьте, что в main-процессе включено открытие внешних URL через `shell.openExternal`.
- Перезапустите приложение после изменений main-процесса.
